Detailed Description

A sensor for collisions on the bottom of b2Body's.

The CollisionDispatcher will call begin_collision and end_collision of this entity. This class then increments/decrements a private member if the foot was subject to the collision.

E.g. the method is_on_ground() can then be used to determine if a player is able to jump.
