Detailed Description

DebugDraw draws the physical representation of Box2d objects.

To help with debugging this utility class outlines all Box2d shapes.

Any use of a DebugDraw object must respect the preprocessor define ENABLE_DEBUG_DRAW, e.g.: 

 #ifdef ENABLE_DEBUG_DRAW
   DebugDraw drawer(..);
 #endif
