Config allows retrieval of user supplied (vm) and internal key/value pairs (Registry).

template float Game::Config::get< float > ( const std::string &  key  )  const [inline]

Get a value for a key (template method).

If the given key exists in the Registry, its value will be returned. Else the value will be retrieved from the variables_map (Config::vm). If the key doesn't exist at all an exception will be thrown.

T must be bool, int, float or std::string.
